# Task 3 – SQL Basics

## Dataset
Superstore sales dataset imported into MySQL.

## Queries Performed

1. Filtered sales data where category = 'Technology'.
   - Purpose: To analyze technology product sales.

2. Aggregated total sales by region using GROUP BY.
   - Output saved as sales_by_region.csv

3. Used HAVING clause to show regions with total sales > 100000.
   - Result shows high-performing regions only.

## Files Included
- queries_task3.sql – All SQL queries
- sales_by_region.csv – Exported query output
